# Описания
 Библиотека с авторами, книгами, изображениями, издетелями связнные с базой данныз sqlite3

# Установка
1. скачать, распаковать
2. установить зависимости, (pip install -r requirements.txt)

# Запуск
1. открыть консоль и перейти в директорию с файлом manage.py
2. запустить сервер коммандой python manage.py runserver

# Работа с базой данных
1. зарегестрировать администратора в консоле (python manage.py createsuperuser)
2. открыть сервер добавить /admin (localhost:8000/admin)
3. ввести логин и пороль
4. вы получили возможность добавить/обновить/удалить данные
